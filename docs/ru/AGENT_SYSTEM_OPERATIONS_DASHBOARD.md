# Панель оператора системы агентов

## Назначение
Это первая operator entrypoint для локальной multi-agent QA system.
Используй ее, чтобы понять blocker, выбрать правильную команду и перейти к нужному runbook.

## Первые команды
- `doctor`
- `health`
- `preflight`
- `index`
- `finalize`
- `watch`

## Правило
Если причина проблемы неясна, начинай с dashboard, а не с редактирования скриптов.
