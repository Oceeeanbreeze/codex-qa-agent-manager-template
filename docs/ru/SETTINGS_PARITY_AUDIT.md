# Аудит паритета настроек

## Что было добавлено
В публичный шаблон были добавлены настройки, которые раньше были только во внутренней системе:
- coordinator responsibilities
- richer role definitions
- memory search и memory capture policy
- milestone archival policy
- checkpoint и anti-loop rules
- model escalation и role profiles
- richer route heuristics
- detailed memory include paths and query hints
- public `memory/ROLE_TOOLING.md`
- public `configs/role-profiles.template.yaml`

## Что намеренно не публикуется
- live vault contents
- generated indexes
- private runtime scripts с локальными зависимостями
- локальные пути
- внутренние incidents, screenshots и logs
- credentials и secrets
